I want you to develop a simple banking project in Java. Name of the project: "Ayvalık Bank CC-1"
Use Java 25. 
There are mainly two sets of functionalities: Customer and Account. Each customer may have one or more accounts. 
There are only two actors: admin and customer
Admin can create a customer, delete a customer, and list customers
Customers can create accounts, deposit and withdraw money and learn the balance in an account. They also may want to  get all transactions made to an account. The accounts may have balance in three currencies: USD, EUR and TL.
Customers can change their passwords. Each password should have at least 8 characters at most 16 characters. Passwords should contain at least on ecapital case one lower case letter, one digit and one special characters such as ! or *. Customers may change their passwords. Make sure that new passwords are all valid and should not be the same as last three passwords used by the customer.
Customers should also be able to list their accounts. Customers should also be able to transfer money among accounts. Transfering money amount the accounts of the same customer is free. But transfering money amount the accounts of the different customers incur a fee, which is set by the admin.

Use PostfreSQL as a database.

Architecture: Use Hexagonal Architecture. Create necessary ports and adapters. All dependencies must be from outside into the domain layer. Use can create domain services, entities and value objects in the domain layer.
You can create value objects as records.
Enforce good encaplsulation. Avoid using setters and getters on entities, avoid anemic domain objects, instead make
entities and value objects rich in terms of business. Avaoid using any framework in domain layer. So do not annotate entities with JPA or Spring's annotations. Create DTOs for entities to use in JPA. DTOs should reside outside the domain layer.
Create all in and out adapters.
Application has also application services that is responsible for non-domain functionality such as interacting with the database.

Use following technologies: Spring Boot, JPA, PostgreSQL
Provide all REST API endpoints.
Create tests using JUnit and Mockito
Use maven to manage the lifecycle of the project.
Use the root package: dev.kaldiroglu.hexagonal.ayvalikbank
All other packages must go under this root package.
